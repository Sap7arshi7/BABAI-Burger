# 🍔 Babai Burger - E-Commerce Website

A fully functional restaurant e-commerce website with shopping cart, bKash payment integration, and customer review system.

## 🌟 Live Demo
(https://sap7arshi7.github.io/BABAI-Burger/)

## ✨ Features

- 🛒 **Shopping Cart** - Add/remove items, update quantities
- 💚 **bKash Payment** - Mock payment with validation
- ⭐ **Review System** - Rate burgers and write reviews
- 💾 **Local Storage** - Cart and reviews persist after refresh
- 📱 **Responsive Design** - Works on mobile, tablet, and desktop
- 🍔 **6 Burger Options** - Real images and descriptions

## 🛠️ Technologies Used

| Technology | Purpose |

| HTML5 | Structure |
| CSS3 | Styling & Animations |
| JavaScript | Interactivity & Logic |
| LocalStorage | Data persistence |
